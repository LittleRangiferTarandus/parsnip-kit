# kebabCase
      
把字符串转换到短横线命名。

### Usage

```ts
camelCase('HelloWorld') // 'hello-world'
```

      
### Arguments
      
| Arg | Type | Optional | Default | Description |
| --- | --- | --- | --- | --- |
| `-` | `string` | `false` | `undefined` | `待转换的字符串` |
      
### Returns

| Type |
| ---  |
| `string`  |
