# splitToWords
      
把用于命名的字符串拆分为单词。

### Usage

```ts
splitToWords('-_i need XmlHTTPRequest -_') // ['i', 'need', 'Xml', 'HTTP', 'Request']
```

      
### Arguments
      
| Arg | Type | Optional | Default | Description |
| --- | --- | --- | --- | --- |
| `-` | `string` | `false` | `undefined` | `待转换的字符串` |
      
### Returns

| Type |
| ---  |
| `string[]`  |
