# capitalize
      
把单词变成首字母大写的形式。

### Usage

```ts
capitalize('HelloWorld') // 'Helloworld'
```

      
### Arguments
      
| Arg | Type | Optional | Default | Description |
| --- | --- | --- | --- | --- |
| `-` | `string` | `false` | `undefined` | `待转换的字符串` |
      
### Returns

| Type |
| ---  |
| `string`  |
