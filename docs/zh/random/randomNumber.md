# randomNumber
[[[desc randomNumber
返回随机数字，取值范围为 [`start`, `end`)。
]]]
[[[version randomNumber
  
]]]
### Usage

```ts
import { randomNumber } from 'parsnip-kit'

randomNumber() // a number in [0, 1)
randomNumber(0, 100) // a number in [0, 100)

```


### API

#### Arguments
[[[params randomNumber
start:区间左边界
end:区间右边界
]]]
#### Returns
[[[returns randomNumber

]]]
