# isString
      
判断入参是否为字符串。

### Usage

```ts
isString('test') // true
isString(123) // false
```

      
### Arguments
      
| Arg | Type | Optional | Default | Description |
| --- | --- | --- | --- | --- |
| `arg` | `any` | `false` | `undefined` | `待判断的入参` |
      
### Returns

| Type |
| ---  |
| `boolean`  |
