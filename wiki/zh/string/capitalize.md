# capitalize
![Static Badge](https://img.shields.io/badge/Coverage-100.00%-FF8C00)
      
把字符串变成首字母大写的形式。

> Added in v0.0.1



### Usage

```ts
import { capitalize } from 'parsnip-kit'

capitalize('HelloWorld') // 'Helloworld'
```


### API

#### Arguments

| Arg | Type | Optional | Default | Description |
| --- | --- | --- | --- | --- |
| `arg` | `any` | `false` | `undefined` | 待转换的字符串  |

#### Returns

| Type |
| ---  |
| `string`  |
