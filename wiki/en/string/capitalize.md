# capitalize
      
Capitalize the first letter of the string.

### Usage

```ts
capitalize('HelloWorld') // 'Helloworld'
```

      
### Arguments
      
| Arg | Type | Optional | Default | Description |
| --- | --- | --- | --- | --- |
| `arg` | `string` | `false` | `undefined` | `The string to be converted.` |
      
### Returns

| Type |
| ---  |
| `string`  |
