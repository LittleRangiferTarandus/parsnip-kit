# kebabCase
      
Convert the string to kebab-case.

### Usage

```ts
camelCase('HelloWorld') // 'hello-world'
```

      
### Arguments
      
| Arg | Type | Optional | Default | Description |
| --- | --- | --- | --- | --- |
| `arg` | `string` | `false` | `undefined` | `The string to be converted.` |
      
### Returns

| Type |
| ---  |
| `string`  |
